How to Export from MetaTrader
MetaTrader 4:
  1. Open MT4 and press Ctrl+T to open the Terminal
  2. Click the Account History tab
  3. Right-click and select your date range
  4. Right-click again and choose Save as Report
  5. Save as HTML file
MetaTrader 5:
  1. Open MT5 and press Ctrl+T to open the Toolbox
  2. Click the History tab
  3. Right-click, set date range, then Export
  4. Choose HTML format and save
Supported data: Closed trades with symbol, type, lot size, prices, commission, swap, and profit.

Note: Balance entries, deposits, and withdrawals are automatically excluded. The uploaded file is not stored, only the extracted trade data is saved to your journal.
